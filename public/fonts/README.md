# Bornomala Font Files

Place the Bornomala Unicode Bengali font files here:

- `Bornomala.woff2`
- `Bornomala.woff`
- `Bornomala-Bold.woff2`
- `Bornomala-Bold.woff`

Download the font from the official source and convert to woff/woff2 if needed.
Once the files are placed here, the Bengali text across the app will automatically use Bornomala.
